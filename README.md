### WebSocketForiOS

##### WebSocket for iOS

本示例程序使用了第三方库[`SocketRocket`](https://github.com/square/SocketRocket)，实现了iOS客户端与服务器之间的WebSocket连接。

>##### Tip:
>测试前请创建一个WebSocket Server，推荐使用nodejs。
